# Flunk

The Flutter cyberpunk app. Built for the [flutter hackathon](https://flutterhackathon.com) #Hack20.

---

## Building

`master` channel is recommended

- Clone the repo
- Setup your signing keys
- Change the secrets
- Run build.sh

---

## Download

A release build is published on GitHub releases
